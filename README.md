# 🚗 Used Cars Data Wrangling

## 📖 Overview

This project demonstrates the data wrangling process for a used cars dataset using Python and Pandas. It is part of the IBM Data Analysis with Python course and focuses on preparing raw data for analysis and machine learning.

---

## 🎯 Objectives

- Handle missing values
- Correct data types
- Standardize numerical data
- Normalize data
- Create binned features
- Generate indicator (dummy) variables
- Export the cleaned dataset

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 📂 Dataset

- **Dataset:** Automobile Dataset
- **Source:** IBM Skills Network (Data Analysis with Python)

---

## 📈 Data Wrangling Tasks

- Missing value treatment
- Data type conversion
- Data standardization
- Data normalization
- Data binning
- One-Hot Encoding (Indicator Variables)
